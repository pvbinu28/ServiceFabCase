Hello, world!

Welcome to your new single-page application, built with:

  • ASP.NET Core and C# for cross-platform server-side code
  • Angular and TypeScript for client-side code
  • Bootstrap for layout and styling
  • Service Fabric - Both API and Process Service running on top of Service Fabric

Application Layers

  • UI - UI runs on asp.net core with angular as a stateless service fabric service.
  • Processor API - Processor API is asp.net core API running as Stateful service fabric application
  • Database - MongoDB is used for storig data. all data is stored in Reliable collection and synced with MongoDB

To run the code install service fabric sdk and setup a local cluster, then install mongo DB and updated the DB connection string.
